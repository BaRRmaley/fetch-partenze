Fetches the nearest trains from viaggiatreno.it and allows to integrate the result to Fully kiosk browser.
In my case the script is made for departures from Ivrea, Italy. But Ivrea station code can be changed to any of the stations, available by search request in the following format
http://www.viaggiatreno.it/infomobilita/resteasy/viaggiatreno/cercaStazione/Ivrea (change city name to yours)
Unfortunately, viaggiatreno is not 100% stable, but it still works usually :)
